# PoC: 喫煙禁止エリア違反検知システム

## 概要

路上喫煙禁止条例の施行エリアに専用カメラを設置し、歩きタバコをリアルタイム検知するシステムの実証実験計画。

## 対象エリア候補

| エリア | 条例 | 過料 | カメラ設置可能性 |
|---|---|---|---|
| 千代田区（秋葉原・神保町） | 路上喫煙禁止条例 | 2,000円 | 商店街の既設ポールに設置 |
| 港区（新橋・品川） | 歩きたばこ禁止 | 過料なし | 駅前広場のポールに設置 |
| 渋谷区（渋谷駅周辺） | 路上喫煙禁止 | なし | センター街入口に設置 |

推奨: **千代田区**（過料あり＝効果測定しやすい、秋葉原は歩行者密度高い）

## システム構成

```
専用カメラ (1080p, PoE)
  │
  ▼
エッジPC (Jetson Orin / RTX GPU)
  │ ├── YOLOv8 人物検出+追跡
  │ ├── タバコ検出モデル (fine-tuned)
  │ ├── YOLOv8-pose (手-口パターン)
  │ └── VLM検証 (false positive除外)
  │
  ▼
管理ダッシュボード
  │ ├── リアルタイムアラート
  │ ├── 検知履歴・統計
  │ ├── ヒートマップ
  │ └── CSV/PDFレポート出力
  │
  ▼
通知
  ├── 巡回員スマホへプッシュ通知
  └── 管理事務所モニター表示
```

## カメラ要件

| 項目 | 仕様 |
|---|---|
| 解像度 | 1920x1080 (Full HD) 以上 |
| フレームレート | 30fps |
| 設置高さ | 3〜5m（歩行者の上半身が大きく映る） |
| 画角 | 水平60〜90度 |
| 接続 | PoE (Power over Ethernet) |
| 耐候性 | IP66以上 |
| 推奨機種 | Axis M2036-LE / Hikvision DS-2CD2T47G2 |

## 設置イメージ

```
        カメラ (高さ4m)
          │
          ▼ 画角70度
    ┌─────────────────┐
    │  検知エリア       │
    │  (幅10m x 奥行5m) │
    │                   │
    │  歩行者 ← → 歩行者 │
    │                   │
    └─────────────────┘
         歩道/商店街
```

1カメラあたり約50㎡をカバー。100mの商店街なら10台で全域カバー。

## 期待精度 (1080p 専用カメラ)

## 効果測定指標 (KPI)

1. **検知精度**: Precision / Recall / F1 スコア
2. **false positive率**: VLM検証後の誤検知率 <10%
3. **巡回効率**: 検知→巡回員到着の平均時間
4. **抑止効果**: PoC前後の喫煙者数の変化
5. **過料徴収**: 検知→指導→過料の件数

## 法的考慮事項

- 肖像権: 検知スナップショットは社内利用のみ、外部非公開
- 個人情報: 顔にモザイク処理、一定期間後自動削除
- 条例根拠: 千代田区生活環境条例第25条（路上喫煙禁止）
- カメラ設置: 防犯カメラ運用ガイドライン準拠
- データ保持: 検知データは30日間保持後削除

## 拡張計画
